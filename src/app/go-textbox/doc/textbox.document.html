Description

{{componentDescription}}

Feature
  • validate ข้อมูลใน textbox ได้ตาม textbox type
  • ส่งค่าข้อมูล และ ผลการ validate เพื่อนำไปให้ต่อได้
  • ใช้งานร่วมกับ form validation ได้
  • ระบุรูปแบบข้อมูลของ textbox type currency ได้
  • ระบุ regular expression ที่ต้องการใช้ validate ข้อมูลได้
  • กำหนดให้เป็น textbox ที่ถูกปิด(disable) หรืออ่านได้อย่างเดียว(readoonly) ได้
  • กำหนดความยาวสูงสุดในการรับข้อมูลได้
  • ใส่สีให้กับ textbox ได้
  • กำหนดข้อความเตือน และตำแหน่งของข้อความได้
